## Solution Approach

### Q1 - Odd or Even
- Accepted a number as input.
- Used the modulus (`%`) operator to check divisibility by 2.
- Printed whether the number is odd or even.

### Q2 - Factorial
- Accepted an integer from the user.
- Used a loop to multiply numbers from 1 to n.
- Displayed the factorial.

### Q3 - Fibonacci Series
- Implemented the series using both `for` and `while` loops.
- Printed the first n terms of the sequence.

### Q4 - Prime Number
- Checked divisibility from 2 to √n (or n-1 if that's what you implemented).
- Printed whether the number is prime.

### Q5 - Compare Two Files
- Compared the contents of two files.
- Removed duplicate content if found.
- Displayed the final result.

### Q6 - Palindrome
- Reversed the given number.
- Compared the reversed value with the original number.

### Q7 - Multiplication Table
- Used a loop to generate the multiplication table for the given number.

### Q8 - Log Finder
- Recursively searched for `.log` files modified within the last 7 days.
- Counted the total lines.
- Displayed a sorted report of files and their line counts.

### Q9 - System Report
- Gathered system information such as CPU, memory, disk usage, uptime, and top processes.
- Saved the report with a timestamped filename.

### Q10 - Password Generator
- Generated secure random passwords.
- Allowed configurable password length and options.
- Used Linux randomness sources for security.
